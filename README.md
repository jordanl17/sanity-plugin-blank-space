# sanity-plugin-blank-space

Turborepo monorepo containing the `sanity-plugin-blank-space` Sanity Studio plugin.

## Packages

| Package                                                                       | Description                                                   |
| ----------------------------------------------------------------------------- | ------------------------------------------------------------- |
| [`sanity-plugin-blank-space`](./packages/sanity-plugin-blank-space/README.md) | Plugin that renders a custom component in the empty pane area |

## Development

```sh
pnpm install
pnpm dev:studio    # starts Sanity Studio at http://localhost:3333
```
